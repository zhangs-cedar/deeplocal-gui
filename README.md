# deeplocal-gui

深度学习本地 GUI 训练工具 - PyQt6 版本

## 架构

```
项目中心
├── 项目列表（左侧）
└── 项目详情（右侧）
    ├── 项目信息
    ├── 工作区列表
    └── 进入工作区
```

## 运行

```bash
python app.py
```

## 功能

### 已实现
- ✅ 项目中心界面
- ✅ 项目列表展示
- ✅ 项目创建
- ✅ 项目详情显示
- ✅ 工作区创建
- ✅ 数据持久化（JSON）

## 依赖

```bash
pip install PyQt6
```

## 数据存储

项目数据存储在：`~/.deeplocal-gui/projects/`

每个项目包含：
- `project.json` - 项目元信息
- `workspaces/` - 工作区目录
